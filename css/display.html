1. display: block

.block
.block

– Each `.block` starts on a new line and stretches the full available width (or your set width).

2. display: inline

.inline .inline

– `.inline` boxes flow with text. Width/height settings are ignored.

3. display: none

.none (you won’t see me!)
I’m still visible

– `.none` removes the element entirely from the layout.

4. display: inline-block

.inline-block .inline-block

– Like inline, but you can set width/height.

5. display: table (with rows & cells)

Cell 1
Cell 2
Cell 3
Cell 4
Cell 5
Cell 6

– Mirrors HTML `

` structure via CSS. Useful when you need table-like layouts without actual `
` markup.

6. display: flex

Flex 1
Flex 2
Flex 3

– Creates a one-dimensional flex container. Children share a row (or column if you set `flex-direction`).

7. display: grid

– Creates a two-dimensional grid. You define rows/columns via `grid-template-...`.
